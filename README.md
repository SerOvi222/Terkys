# Visual Maker

这是一个低代码平台搭建的项目，可以通过一些简单的拖拽和配置生成页面

- 💪 Vue 3 Composition API
- 🔥 Written in TypeScript
- 🎨 Style based on Element-plus
- 🍍 Managing state with Pinia

# 本地启动

---

```bash
git clone https://github.com/luoyangC/visual-maker.git
cd visual-maker
yarn
yarn run start
```
